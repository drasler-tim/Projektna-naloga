# Projektna-naloga

Moje ime je Tim. Sem študent finančne matematike in to je moja projektna naloga pri predmetu Uvod v programiranje. Iz spletne strani [EV Avtomobili](https://ev-database.org/#group=vehicle-group&rs-pr=10000_100000&rs-er=0_1000&rs-ld=0_1000&rs-ac=2_23&rs-dcfc=0_400&rs-ub=10_200&rs-tw=0_2500&rs-ef=100_350&rs-sa=-1_5&rs-w=1000_3500&rs-c=0_5000&rs-y=2010_2030&s=1&p=0-50) sem pobral 1140 podatkov, od tega za vsak podatek id, naslov, model, doseg, kapaciteta baterije, učinkovitost, hitro polnenje, teža , vleka, pospešek, volumen prtljažnika, doseg z enim postankom, cena na kilometer dosega in cena avtomobila.

## Navodila za uspešno delovanje programa

1. Pojdite na spletno stran [EV Avtomobili](https://ev-database.org/#group=vehicle-group&rs-pr=10000_100000&rs-er=0_1000&rs-ld=0_1000&rs-ac=2_23&rs-dcfc=0_400&rs-ub=10_200&rs-tw=0_2500&rs-ef=100_350&rs-sa=-1_5&rs-w=1000_3500&rs-c=0_5000&rs-y=2010_2030&s=1&p=0-50)
2. Pritisnite <kbd>Ctrl</kbd> + <kbd>U</kbd>, da se vam v novem zavikhu brskalnika odpre okno izvorne kode HTML
3. S bližnjico <kbd>Ctrl</kbd> + <kbd>A</kbd> izberite vse in s <kbd>Ctrl</kbd> + <kbd>C</kbd> kopirajte
4. Odprite Visual Studio Code in v željeni mapi ustvarite novo datoteko z imenom "avti.html"
5. Vanjo kopirajte izvorno kodo s bližnjico <kbd>Ctrl</kbd> + <kbd>V</kbd>
6. Nato si namestite datoteke iz mojega repozitorija in jih poženite v Visual Studio Code-u

## Analiza podatkov

Izluščeni podatki so zapisani v CSV datoteki in nato uporabljeni za analizo v jupyter nootbook-u.
